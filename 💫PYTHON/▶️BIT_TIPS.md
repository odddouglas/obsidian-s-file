# ▶️不用写分号
# ▶️变量无类型,其存储的数据才有类型 
# ▶️讲究一个直接输出
# ▶️万物字符串转换但反之要慎重
# ▶️命名规则_下划线可开头数字不可开头
# ▶️'单'双'三'引号均可定义字符串
# ▶️布尔(0和1)(True和False)
# ▶️语句无括有引:靠缩进判断归属
# ▶️在条件判断中直接input()节省代码
# ▶️for无法构建死循环,待处理数据不可无限大
# ▶️大循环的一次需要等小循环执行一轮
# ▶️ctrl+k+c 行注释 ctrl+shift+/ 块注释
# ▶️shift+alt+mouse拖动
# ▶️return语句后的代码不执行
# ▶️if后面的字符串只要不为空就为True
# ▶️函数可嵌套
# ▶️for的临时变量和函数的局部变量很像
# ▶️适应input语句的括号提示语技巧
# ▶️容器(数组)支持反向下标索引(-1开始)
# ▶️index查不到被嵌套的数据
# ▶️remove方法可通过循环进行批量删除
# ▶️但注意remove方法在list中的内存收缩
# ▶️W3C网站可查询python方法
# ▶️无需记忆函数方法,常用的了解即可
# ▶️单元素元组中的那个数据后面要加逗号
# ▶️元组嵌套的列表依然可以修改
# ▶️字符串中空格也算字符
# ▶️元组和字符串的方法只是得到新变量
# ▶️序列步长为负时,首尾两个下标需要换位
# ▶️序列切片条件省略取结尾,不省略则不取
# ▶️空{ }是空字典不是空集合
# ▶️列表和集合和字典的方法都可以直接操作
# ▶️集合和字典要用for进行遍历
# ▶️字典嵌套时需要注意不要漏掉逗号
# ▶️容器被str(),将连同符号一起变成字符串
# ▶️字符串被强制转换时,字符会被拆开
# ▶️字符大小比较对应ASCII码值的比较
# ▶️交互界面需要定义完函数后再换一行
# ▶️( / ) 被除数和除数都为整,也得到浮点型
# ▶️( // ) 两个操作数实数即可
# ▶️a,b=b,a可以实现无temp交换
# ▶️元组存储的是"数据的引用"而非数据本身
# ▶️逗号分隔的数据都会被系统默认为元组
# ▶️input输入int和float时记得强制转换
# ▶️r后后面的内容将会原始输出
# ▶️特殊转义\%% 会输出一个百分号%
# ▶️函数外部的赋值语句就是创建全局变量
# ▶️函数直接返回条件判断语句,即返回bool类型
